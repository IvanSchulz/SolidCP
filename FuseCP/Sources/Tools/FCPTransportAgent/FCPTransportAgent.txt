FCP Transport Agent Installation
================================

Perform the following steps:

A) Copy the files FCPTransportAgent.dll and FCPTransportAgent.dll.config to "C:\Program Files\Microsoft\Exchange Server\V14\Public"
B) Import the FCPTransportAgent.reg to create the event source
C) Use the registry editor and provide the "NETWORK SERVICE" Full Control on the following Key
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\eventlog\FCPTransportAgent
D) Run the following powershell command in the exchange management shell:
Install-TransportAgent "FCPTransportAgent" -TransportAgentFactory FCPTransportAgent.FCPRoutingAgentFactory -AssemblyPath "C:\Program Files\Microsoft\Exchange Server\V14\Public\FCPTransportAgent.dll"
E) Enable-TransportAgent "FCPTransportAgent"
F) Restart-Service MSExchangeTransport
